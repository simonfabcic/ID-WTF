# ID-WTF

Application for adding notes, which you don't want to forget (I don't want to forget)

## Features

Adding the ID-WTFs, which contains:

-   Title
-   Source
-   Content
-   Timestamp (auto add)

## Implemented later

-   Tags (custom user tags)
-   Export
-   Search
-   Share
